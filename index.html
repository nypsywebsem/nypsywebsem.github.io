Workshop summary

We will be meeting on Thursday, October 19th, in Meyer 551 for a workshop for those interested in making their own free, beautifully designed website (for yourself, lab, etc.). This will be hands on, and I sincerely promise everyone who attends will have a solid website up and running by the end of the workshop, with knowledge necessary to further update and customize it thereafter. Intro and getting everyones site template up will only take an hour, and then we will stay in the room and help everyone build and edit their websites for a couple hours.

In only a couple hours, with the methods/templates we will use, your website can look like or unlike any one of the examples above, and many more, taken and highly customized from these templates from HTML5 UP.

These websites will be free, beautifully designed, compatible across all browsers/devices, and require very little work on your end. I will also cover how to customize these easily (colors, text format, etc.). These may easily be moved to other host locations as well (eg, PIs/labs could likely get NYU to host), including your own purchased domain if you wish.

Getting started

  1. Choose your design! Pick a template from HTML5 UP to work with. Or Jekyll if you are interested in blogging. Or one from somewhere else! I personally recommend these templates for many reasons, such as ease, formatting, and importantly, compatibility across devices and browsers. Github Pages, which we will use to host our websites during the workshop for free, also offers templates.
  2. Get your site hosted and live! Set up a github account, and page, by following the instructions on the Github Pages home.
  3. Upload and update as you go! To upload your html pieces and update your github page, as you work on your local repository, use the following options:
    • Use command line to submit changes (if you are not using the Github GUI app for Windows or Mac):
      $ git add -A; git commit -am "update"; git push origin master
    • Use github's app or website interactively to upload and update files
